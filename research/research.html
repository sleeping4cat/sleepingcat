Project Highlights

A New Type of AlexNet - WasteNet Recreation

AlexNet is widely considered one of the best large Neural Network models, especially in Computer Vision. I've recreated WasteNet, a new Image Classification Model developed by UK Government researchers, in Tensorflow. This project leverages fine-tuning and Hybrid Tuning for faster and better image classification results, preventing destruction of information in the fine-tuning stage.

Link: https://github.com/sleepingcat4/WasteNet

GUI for Object Detection (YOLO v5) - WasteClassification

I've developed a personalized GUI for local Object Detection tasks. Using Flask as the backend, this project provides a light-weight inference interface with outstandingly simple and well-documented code.

Link: https://github.com/sleepingcat4/wasteclassification
